Yaboigoku blog built with Laravel

Register page

Here the user registers to use the blog

Login page

Here the user logs in the use the blog

Home Page Logged out

This is the home page available to users who are currently logged out here the user can only Start Writing(Create a Post) after logging in.

Home Page Logged in

This is the home page the user is redirected to after successfully logging in or registering, here the user can Start Writing(Create a Post) since the user is logged out.

Create Post

Here the user has to be logged in to create a post.

All Posts with Search

This page reveals all the posts available, its available to all users(logged in or logged out) users can also search for specific posts as well.

Single Post Logged in

This page shows a single post, here the user is logged in so the user can comment on the post.

Single Post Logged out

This page shows a single post, here the user is logged out so the user cannot comment until the user logs in.

Register page

Here the user registers to use the ecommerce website

Login page

Here the user logs in the use the ecommerce website

Home Page Logged out

This is the home page available to users who are currently logged out comment section is available too.

Home Page Logged in

This is the home page the user is redirected to after successfully logging in or registering, here the user can access their orders, cart, and the admin panel comment section is available too.

Here users can search for specific products.

Adding to cart

Here users just have to hover over a product and adjust the parameters to add to their cart

Cart

This page reveals the user's cart

Orders

Here users can track their orders and can also cancel their orders as well

Stripe payment page

This is a payment channel using stripe for the order payment.

Admin Panel

Admin (All Products)

Here the admin can add a new product to be listed for sale.
